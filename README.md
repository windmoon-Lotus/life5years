# 人生五年访谈计划 (Five Years of Life)

> 记录普通人的变化、坚持，以及如何被生活塑造。

## 项目简介

“人生五年”访谈计划是一个尝试借鉴纪录片《人生七年》（Seven Up!）的记录方式，专注于记录普通人真实生命轨迹的长期访谈项目。

在变化越来越快的今天，我们很难找到一个稳定不变的支点。即使是曾经被视为人生重心的“职业”，也可能会随着时代、环境和个人选择不断发生改变。因此，本项目不以单一的职业或身份来定义一个人，而是将关注点回归到“人”本身和“人生的变化”上。

本计划的核心主旨是记录：**“你是谁，你如何生活，你如何变化”**。

## 项目愿景

通过长期的追踪访谈，记录不同个体在同一时代中的生活样貌，在时间的流动中真正观察一个人如何成长、如何思考。我们不承诺给出人生的标准答案，而是希望提供足够多、足够真实的“路径样本”，扩宽我们对人生的认知，让每一位受访者和阅读者都能以更笃定、更平和的心态去完成一生的生活选择，用积极的心态迎接未来的人生之旅。

## 访谈规划

- **更新频率**：计划每两周访谈一位受访者。
- **样本规模**：累计记录一百多人。
- **回访机制**：每隔五年对受访者进行一次循环回访，记录时间跨度内的变化。

## 访谈内容模块

访谈将立足“当下”，慢节奏展开，尊重受访者的意愿与边界：

1. **当下的生活状态**：处世态度、做人原则、价值观、生活习惯、消费观、金钱观、人际交往的准则，以及目前真正想追求的生活。
2. **亲密关系与家庭**：对恋爱与婚姻的看法、当前的状态，以及对子女教育的思考。
3. **当下的工作状态**：工作不再是定义一个人的核心，而是作为此刻生活的一部分来被探讨。
4. **回望与畅想**：
   - 过去：童年、求学经历、家庭、家乡，以及那些塑造了受访者的人和事。
   - 未来：畅想五年后的自己与生活。

## 目录结构说明

- **`人生五年访谈计划.md`**：项目发起初衷、计划思路说明。
- **`/访谈准备资料`**：访谈所需的思路清单、框架表等。
- **`/人生话题`**：对于人生各个核心话题（如婚姻观、人生的意义、人生几道坑）的独立思考与归纳。
- **`/随笔`**：项目的日常记录、访谈复盘记录与相关的读书笔记。
- **`/[日期]-[受访者特征]`**：按受访者和日期归档的访谈记录文件夹（包含访谈音频、AI转录文本、提纲及最终发布的文章）。

## 参与与授权

本项目的所有访谈录音及后续使用均在受访者完全同意的前提下进行。初步资料整理完成后，将交由受访者确认，无误后再进行发布。

同时，受访者对自己的访谈资料享有永久的撤回权力（若资料已被非本项目人员使用，因精力有限可能无法协助长期跟进，敬请谅解）。
